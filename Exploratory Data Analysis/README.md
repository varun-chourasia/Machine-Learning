## Purpose

Exploratory Data Analysis is crucial for:

- **Data Discovery:** Initial examination of dataset structure, size, and composition
- **Statistical Summaries:** Descriptive statistics, data types, and distribution analysis
- **Data Quality Assessment:** Identification of missing values, outliers, and data anomalies
- **Visualization:** Creation of charts and plots to understand patterns and relationships
- **Feature Understanding:** Examination of individual variables and their relationships
- **Insights Generation:** Discovery of key patterns that inform preprocessing and modeling decisions

## Recommended Workflow

1. **Data Loading:** Import datasets from the `data/` folders in other directories
2. **Dataset Overview:** Display basic information (shape, data types, head, describe)
3. **Missing Data Analysis:** Check for null values and plan appropriate handling strategies
4. **Distribution Analysis:** Visualize histograms, box plots, and density distributions
5. **Correlation Analysis:** Generate correlation matrices and heatmaps
6. **Outlier Detection:** Identify and document potential anomalies
7. **Documentation:** Record key findings and insights for subsequent phases
